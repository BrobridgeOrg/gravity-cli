# Gravity CLI

Gravity CLI is a command line utility for managing gravity cluster.

## Build

``` bash
go build
```

---

## Usage

Currently, Gravity CLI supports:

* Product Management
* Rule Management
* Token Management

### Publish event

```shell
gravity-cli pub accountCreated '{"id":4,"name":"fred"}'
```
